# Sonrisa Argentina — Marketing Digital Dental

Agencia de marketing digital especializada en clínicas dentales en Argentina. Más pacientes nuevos con SEO local, sitios web profesionales y gestión de reseñas.

## Stack

- **Framework:** Next.js 14 (App Router)
- **Estilos:** Tailwind CSS
- **Animaciones:** GSAP
- **Contenido:** MDX
- **Deploy:** Vercel

## Desarrollo local

```bash
npm install
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000).

## Build de producción

```bash
npm run build
npm start
```

## Deploy

El proyecto se despliega automáticamente en Vercel al hacer push a `main`.
